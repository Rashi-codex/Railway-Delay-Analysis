# Railway-Delay-Analysis
 Project Description

This project analyzes railway train delays using data analysis techniques.
The aim is to understand why trains get delayed, how often delays occur, and which routes or trains are most affected.

The analysis is done using a Jupyter Notebook (.ipynb), making it easy to understand with step-by-step code, outputs, and graphs.

## Project Objectives

Study train delay patterns

Calculate delay duration

Identify frequently delayed trains

Understand route-wise delays

Present insights using graphs

## Dataset Information

The dataset includes:

Train Number / Name

Source Station

Destination Station

Scheduled Arrival & Departure Time

Actual Arrival & Departure Time

Delay Time (minutes)
